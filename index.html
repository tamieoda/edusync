Como configurar o Google Drive

  1. Acesse Google Cloud Console e crie um projeto novo (ou use um existente).
  2. No menu, vá em APIs & Services → Library e ative a Google Drive API.
  3. Vá em APIs & Services → OAuth consent screen, configure como External, preencha os dados básicos e adicione seu email como usuário de teste.
  4. Vá em APIs & Services → Credentials → Create Credentials → OAuth client ID.
  5. Tipo de aplicativo: Web application.
  6. Em Authorized JavaScript origins, adicione a URL onde esta página estará hospedada (ex.: https://seuuser.github.io) ou http://localhost:8000 se for testar localmente.
  7. Copie o Client ID gerado e cole no campo acima.
  8. Clique em Conectar Drive no topo da página.
Importante: o arquivo HTML precisa estar hospedado em uma URL (não abrir como file://) para o OAuth funcionar. Use GitHub Pages, Netlify, ou rode um servidor local com python -m http.server.
